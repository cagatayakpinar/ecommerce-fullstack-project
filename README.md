## Technologies
<ul>
  <li>Front End = Vue.js</li>
  <li>Front End Framework = Vuetify</li>
  <li>Back End = Node.js</li>
  <li>API Server = Server(Express.js), Client(Axios)</li>
  <li>Loging = Winston</li>
  <li>Database = PostgreSQL (pg Pool) with Salt crypted</li>
</ul>
